# README

This repository contains a simple application to generate a .svg file
that can be used to laser cut a coaster with an image.

Instructions on usage and how to laser cut can be found in the following sections.

## Installation

To install, clone the repository, make a virtual environment and install.
```bash
uv venv
source .venv/bin/activate
uv pip install .
```

## Usage

After installing and in the correct environment,
you can run the application using the following command:

```bash
coaster-generator <path-to-image>
```
